ler:InformationsressourceType

Data kommer fra XSD'en (via xmlschema's kompilerede model), ikke fra featurekatalogets docx.

Arvekæde

Viser arvekæden for XSD-typen 'ler:InformationsressourceType', fra den øverste rodtype og ned til denne type selv, samt om hver type i kæden er abstrakt. Kæden bygges ikke ved at læse XSD'ens rå base="..."-attributter direkte, men ved at bruge SISSA's xmlschema-bibliotek (Python), som kompilerer hele skemaet til én samlet model og løser hver types base_type til den faktiske forældretype; vi følger så denne kæde opad, til vi når en type uden nogen forælder.
