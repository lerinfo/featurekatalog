TextAnnotation

Denne side viser, hvad der står om featuretypen 'TextAnnotation' i featurekatalogets docx. Data kommer altså fra docx'en, ikke fra XSD-skemaet.

Definition
textual marking on illustrative material for the purpose of clarification
Kilde
based on [ISO 19117:2012, 4.1]
Type
Featuretype
